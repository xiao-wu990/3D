# 局域网文件传输工具 📡

简单的局域网文件传输工具，支持发送端和接收端双向传输文件。

## ✨ 特点

- 🔄 双向传输
- 📱 二维码连接
- 🌐 局域网传输
- 💾 大文件支持
- 🎨 美观UI

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 使用

#### 接收端：
1. 打开 `http://localhost:3000?role=receiver`
2. 等待发送端连接

#### 发送端：
1. 打开 `http://localhost:3000`
2. 选择"发送端"
3. 点击连接
4. 等待接收端点击"接受"

## 📖 详细使用说明

### 第一步：启动接收端
- 在浏览器打开: `http://localhost:3000?role=receiver`
- 会显示你的IP和二维码

### 第二步：启动发送端
- 在另一个浏览器标签页打开: `http://localhost:3000`
- 选择"发送端"
- 会看到接收端列表

### 第三步：建立连接
- 发送端点击"连接"按钮
- 接收端会看到"连接请求"
- 接收端点击"接受"

### 第四步：传输文件
- 连接成功后双方可以互相发送文件
- 拖拽文件或点击选择

## ❓ 常见问题

**Q: 接收端一进去就显示连接请求？**
A: 这是不正常的，可能是因为之前有未断开的连接。刷新页面应该就好了。

**Q: 点击接受按钮没反应？**
A: 检查浏览器控制台（F12）看是否有错误信息。重启服务器试试。

**Q: 能跨互联网传输吗？**
A: 不能。这是局域网传输工具，只能在同一WiFi或有线网络内使用。

**Q: 获得永久网址后能跨互联网用吗？**
A: 即使获得永久网址，仍然需要在同一局域网内使用。要跨互联网传输，请使用真正的云盘服务。
